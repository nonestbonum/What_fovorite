1. С запуском из контейнера у меня совладать не получилось
2. Для развертывания бд в докере из командной строки запустим контейнер с базой:
docker run --name some-postgres --volume db-data:/var/lib/postgresql/data -e POSTGRES_PASSWORD=qqq -e POSTGRES_DB=vv -p 5434:5432 postgres:12-alpine
3. Для того чтобы отправлять сообщения с почты, нужно ее предварительно настроить и завести пароль, я использовала почту от яндекса, как это сделать и как завести пароль описано тут: https://yandex.ru/support/mail/mail-clients/others.html
4. Для создания таблиц логирования запускаем sql файл, это создаст 2 таблицы, триггеры и повесит эти триггеры на круд операции

Дополнительные зависимости:
spring-boot-starter-mail — для отправки сообщения с подтверждением аккаунта
spring-boot-starter-validation — для валидации данных
spring-boot-starter-freemarker — для использования freemarker, чтобы передавать информацию в ftl файлы из методов
